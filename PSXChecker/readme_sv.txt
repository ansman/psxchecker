PSXChecker version 1.0 av Xager

BESKRIVNING:
PSXChecker låter dig testa pspns funktionalitet.

Du kan testa alla knappar (förutom knappar som hem, volym, etc.), skärmen
genom att leta efter döda/fastnade pixlar, back light bleeding, etc och också test högtalarna.

ANVÄND DETTA PROGRAM PÅ EGEN RISK, var tvungen :)


INSTALLATION:
För att installera kopiera bara PSXCHECKER till ms0:/PSP/GAME/
Du borde då ha ms0:/PSP/GAME/PSXCHECKER/EBOOT.PBP

ms0 är bokstaven för din psp. D, E, F, etc

Om du vill kompilera källkoden själv så måste du ha OSLib förutom PSP SDKn i din PATH, OSLib 2.10 is preferred.

Rekommenderad CPU-hastighet: default (300 mHz)
